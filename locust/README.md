## pythonの設定

自分はasdfでpythonを管理している
以下の方法で最新版が利用できるようにした。

```
asdf install python latest
asdf list python
asdf local python latest
python -V # -> Python 3.11.5
```

## locustのインストール

```
pip install locust
locust -V
```
